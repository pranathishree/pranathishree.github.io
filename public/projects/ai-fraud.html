Naive Bayes Classifier
Applies Bayes' Theorem with conditional feature independence. For each company, computes P(Fraud|Features) using prior probabilities learned from training data. Red-word NLP scoring applied to job descriptions via TF-IDF with Laplace smoothing to avoid zero probabilities.

K-Means Clustering (K=3)
Partitions companies into 3 clusters: High-Risk, Suspicious, and Legitimate. Each company's feature vector is compared to pre-trained centroids using Euclidean distance. The nearest centroid determines cluster assignment. Distance to fraud centroid is normalized as a fraud score.
